# La Barca 2

Sito vetrina statico e responsive per la pescheria **La Barca 2**.

## Contenuti

- Homepage one-page in HTML, CSS e JavaScript
- Marchio della barca rigenerato su fondo trasparente
- Sezioni servizi, pescato, punti vendita e contatti
- Collegamenti rapidi a telefono e Google Maps
- Navigazione mobile e animazioni accessibili

Il sito non richiede build: basta aprire `index.html` o pubblicare la cartella su Vercel.
